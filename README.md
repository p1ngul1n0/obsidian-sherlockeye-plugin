# Sherlockeye OSINT

A plugin that integrates the Sherlockeye OSINT API directly into your note-taking environment.

## Features

- Search emails, phones, domains, usernames, and more
- Automatically create notes from search results
- Support for CPF and CNPJ searches (Brazil)
- Deep search mode for expanded results

## Installation

1. Install from the community plugins list
2. Set your Sherlockeye API key in the plugin settings

## Getting Started

- Highlight any text and right-click to search
- Or right-click a file name to search by its title
- Results will be automatically organized as notes in your vault

## API Key

You need a valid Sherlockeye API key to use this plugin. Get one at [sherlockeye.io](https://sherlockeye.io)
